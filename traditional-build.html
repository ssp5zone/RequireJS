Standard (Modded) + Packages + Built

This page uses the final build file created using updated configurations.

As there are no lazy-loads, to test, you can stop the server and direcly open this HTML file in any browser. Since all the neccesary modules and files are bundled up, this page should work file without a server too.

The build is done using the following command,


                        node node_modules/requirejs/bin/r.js -o public/app/traditional/build.config.js
Note! Devs usually build using appDir: "" and out:"" configs. However this creates an entire ditribution directory. This is good while creating multiple modules or bundles.

Configurations:


                        ({    
                            // Without the base URL, it starts looking for input file 
                            // in mainConfigFile location  
                            baseUrl: ".",

                            // use configurations present here
                            // Notice we have use the configurations of different locations
                            mainConfigFile: ["main-modded.js", "blocks/main.js", "coder/main.js", "sidebar/main.js"],

                            // input file
                            name: "main-modded",

                            // output file name
                            // putting it elsewhere creates copies all file (including node_modules) to it
                            // Hence, let it be here.
                            out: "pack.built.js",

                            // We do not require loaders. Replace them with a stub (dummy) function.
                            // This reduces final build file size as well as ensures no lazy call happens.
                            // Since we used packages, Hence the format became /.
                            // How did I know this name, I checked in the built file.
                            stubModules : ['css/css.min', 'text/text'],
                            
                            // disable uglify for checking
                            optimize: 'none',
                        })
